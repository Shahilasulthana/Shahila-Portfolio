Empower Her

We propose a real-time scream detection system using machine learning to enhance safety. It analyzes audio input, detects distress sounds with SVM and MLP models, and classifies urgency. Upon detecting a scream, it automatically alerts the nearest police station via SMS with the user’s location. The user-friendly Kivy-based interface ensures accessibility on desktop and Android for quick emergency response.

Alcoguard

AlcoGuard is an Arduino-based alcohol detection and engine locking system designed to enhance road safety. It uses an alcohol sensor to detect intoxication levels in the driver’s breath. If alcohol is detected, the system automatically locks the vehicle's engine and sends the current location to the responsible authorities via SMS or call, ensuring timely intervention and preventing potential accidents.

Similarity Detection

This project detects similarity between final year student projects at Telkom University by analyzing titles and descriptions using semantic techniques. It calculates similarity scores, recommends ways to improve uniqueness, and prevents students from repeating existing work. The system also assists in literature review analysis and enables faculty to detect potential plagiarism more efficiently.
